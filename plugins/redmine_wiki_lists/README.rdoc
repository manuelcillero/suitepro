= redmine_wiki_lists

This plugin adds wiki macros that display issue lists.

== Installation

* Download zip file from https://github.com/tkusukawa/redmine_wiki_lists/releases
* Unzip
* Move the unziped folder into the $REDMINE/plugins/
* Restart REDMINE
